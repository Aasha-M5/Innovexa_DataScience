# Innovexa_DataScience
Developed a Student Performance Analysis &amp; Prediction System using Python, Pandas, Seaborn, and Scikit-Learn. Performed data cleaning, EDA, feature engineering, and machine learning model evaluation on 1000 student records. Compared multiple models and achieved 94.8% R² accuracy using Random Forest Regressor for score prediction.
About the Project

Understanding what affects student performance is an important challenge in education. In this project, I analyzed student academic and behavioral data to identify the factors that influence final scores and built machine learning models to predict student performance.

The project covers the complete data science workflow, including data cleaning, exploratory data analysis (EDA), feature engineering, model building, and evaluation. Multiple machine learning algorithms were tested and compared, including Linear Regression, Decision Tree, Random Forest, and Support Vector Regressor (SVR).

After evaluating all models, the Random Forest Regressor achieved the best performance with an R² score of 94.8%, making it the most accurate model for predicting student final scores.

This project helped me gain hands-on experience with Python, data visualization, machine learning, and model evaluation while understanding how data-driven insights can support educational decision-making.
